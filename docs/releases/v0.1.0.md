# Release v0.1.0

Published: 2026-02-12

## Summary

`TaskOrchestratorKit` v0.1.0 is the first open-source release, shipping both Objective-C and Swift task orchestration implementations for app startup workflows.

## Keywords

- iOS startup optimization
- iOS 启动优化
- DAG scheduling
- DAG 调度
- task orchestration
- 任务编排

## What's Included

### Objective-C (`TaskOrchestratorKit`)

- Deterministic task ordering (priority + registration order)
- Dependency graph execution (DAG) with cycle detection
- Serial / Main-thread / Concurrent execution modes
- Idempotent skip with pluggable state store
- Observer callbacks and run result structure

### Swift (`TaskOrchestratorSwift`)

- Actor-based orchestration model
- Async/await execution pipeline
- Configurable per-task and wave timeout controls
- Unified `SW*` API naming
- SPM + CocoaPods integration

### Example App

- Objective-C normal DAG run
- Objective-C cycle detection run
- Swift demo flow using `TaskOrchestratorSwift` as a dependency

## Installation

### CocoaPods

```ruby
pod 'TaskOrchestratorKit', :git => 'https://github.com/AikenCod/TaskOrchestratorKit.git', :branch => 'main'
pod 'TaskOrchestratorSwift', :git => 'https://github.com/AikenCod/TaskOrchestratorKit.git', :branch => 'main'
```

### Swift Package Manager

```swift
.package(url: "https://github.com/AikenCod/TaskOrchestratorKit.git", branch: "main")
```

Product:

- `TaskOrchestratorSwift`
